1214. 波动数列

acwing原题链接

思路：背包模型DP

首先假设 t 为首项，每$i$项添加的为 $d_i$。

那么就有等式$s = t + (t + d_1) + (t + d_i + d_2) + … +(t + d_i +…d_{n-1})$

整理得：$s = nt + (n-1)d_1 + (n-2)d_{2} + … +d_{n-1}$

=> $t = \frac{s - [(n-1)d_1 + (n-2)d_{2} + … +d_{n-1}]}{n}$

因为 $t$为整数那么题目就等价于求 $s$ % $n$ == $[(n-1)d_1 + (n-2)d_{2} + … +d_{n-1}]$ % n 的方案数。

将每一个$d_i$看作物品那么就能将他转化为背包问题。
