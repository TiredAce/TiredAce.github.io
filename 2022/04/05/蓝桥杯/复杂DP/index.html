1050. 鸣人的影分身

acwing原题链接

思路分析 1：

dp状态的表示：dp[i][j]表示用了i点查克拉，制造了j个分身的方案数。
dp状态的转移： 用了k点查克拉转移到了dp[i][j],那么就是从dp[i-k][j-1]转移过来的，状态转移公式dp[i][j] += dp[i-k][j-1]

1047. 糖果

acwing原题链接

dp状态的表示：dp[i][j]表示在1~i中挑选的糖果，价值除k余j最大价值组合。

dp状态的转移：1. 不选当前第i个糖果：那么余j的最大价值为dp[i-1][j]。2.选当前第i个糖果：那么要从状态dp[i-1][(j - w) % k]转移，此时的价值为dp[i-1][(j - w) % k] + w。

1222. 密码脱落

acwing原题链接

思路：

题目要求当初状态下至少要脱落多少个种子 => 等价于求至少还要再脱落多少种子能让字符串变成回文串=>求最长回文串。

dp状态表示：dp[i][j]表示从i~j的子串，最长回文子串的长度为dp[i][j]。

dp状态转移：1. s[i] == s[j]那么最长回文子串的长度为dp[i+1][j-1] + 2。2. s[i] != s[j] 那么分三种情况，情况一：s[i] 与 s[j]不包含在回文串中，状态转移为dp[i+1][j-1]。情况二：s[i] 包含但 s[j]不包含在回文串中，状态转移为dp[i][j-1]。情况三：s[j]包含但s[i]不包含在回文串中, 状态转移为dp[i+1][j]。因为情况三的状态已经被情况一和情况二所包含，因此不需要重复计算。

1226. 包子凑数

acwing原题链接

思路：

用到一个数论的知识点如果所有的包子数互质那么会有最大的凑不出来的包子数，否则，将会是无穷个。

dp数组的状态表示：dp[i]表示能否表示i个包子。

dp数组的状态转移：dp[i] |= dp[i - w]，如果去掉 w 个能表示那么当前能凑出i个。

1070. 括号配对

acwing原题链接

思路：

本题和之前的密码脱落思路相似，求至少要添加多少个字符能使这个BE变为GBE那么题意就等价于要至少删除多少个字符能让这个字符串变为GBE字符串，那就等价于求最长的GBE子串。

注意：和回文串不同的是这题对于像[]()的字符串也视为合法的字符串

dp状态的表示：dp[i][j]表示i~j这段子串中最长的合法子串长度为dp[i][j]。

dp状态的转移：情况一： 如果s[i]与s[j]相配对，那么能从状态dp[i+1][j-1]转移过来，表达式为dp[i+1][j-1]+2。

情况二：如果s[i]与s[j]不匹配，那就将i~j划分为许多区间，分别求每个区间最长的合法字符串的长度，再求和。类似石子合并的划分方式详细的状态转移见代码。
